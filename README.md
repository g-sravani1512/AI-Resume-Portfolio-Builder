# AI Resume & Portfolio Builder

An AI-powered web application that:
- Predicts job category from resume text
- Generates resumes, cover letters, and portfolios
- Customizes content using job descriptions
- Exports documents as PDF, Word, and HTML portfolio websites

## Technologies Used
- Python
- Machine Learning (Scikit-learn)
- NLP
- Streamlit

## How to Run Locally

1. Install dependencies:
pip install -r requirements.txt


2. Run the app:
streamlit run app/app.py


## Features
- Job category prediction
- Resume generation
- Cover letter generation
- Portfolio website generation
- PDF and Word downloads
